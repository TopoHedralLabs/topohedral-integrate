pub fn adaptive_quad_2d<F: Fn(f64, f64) -> f64>(
    f: &F,
    opts: AdaptiveQuadOpts,
) -> Result<AdaptiveQuadResult, OptionsError>
Expand description

Adaptively integrates f over AdaptiveQuadOpts::bounds.

The tolerance is checked per rectangle, so the returned aggregate error estimate may exceed opts.tol. max_depth is validated but is not currently enforced.

§Errors

Returns OptionsError if opts is invalid.
